Excerpt Priority

The Search Matches page displays a portion of text from the user's profile when you mouseover them. You can customise the priority here. The first item will often be the one to show up, but in cases where the user has nothing filled in for it, it will head to the next one, and so on.

Settings Control

The plugin should ordinarily sync with your Google Account automatically, but for backups, reinstallations, and possibly upgrades, you can save and restore your settings.

Simply click the export link to download a txt file containing the information, or use the File Uploader to import settings that you have already saved.
