Matches Page Settings

New Tiles improves upon OkCupid's new "Tiled" design, showing 5 in a row instead of 3, and making other styling tweaks, including better consistency of the "Online" indicator.
Classic is the older style, which displays an excerpt from the user's profile, but otherwise contains much the same information.
Wide Tiles is closer to New Tiles but contains more information. It looks more like the style on the Visitors page.

Excerpt Priority

The "classic" view for the Matches page displays a portion of text from the user's profile. You can customise the priority here. The first item will often be the one to show up, but in cases where the user has nothing filled in for it, it will head to the next one, and so on.

Settings Control

The plugin should ordinarily sync with your Google Account automatically, but for backups, reinstallations, and possibly upgrades, you can save and restore your settings. Simply click the export link to download a txt file containing the information, or use the File Uploader to import settings that you have already saved.
